# 📒 Contact Book (Streamlit)

A Contact Book application built using Python and Streamlit that helps users manage contact information efficiently.

## 🚀 Features

- Add contacts
- View contacts
- Search contacts
- Update contact details
- Delete contacts
- Store contacts using JSON
- Interactive Streamlit interface

## 🛠️ Technologies Used

- Python
- Streamlit
- JSON
- Pandas

## 📂 Project Structure

```
Contact_Book_Streamlit/
│
├── app.py
└── contacts.json
```

## ▶️ How to Run

Install Streamlit

```bash
pip install streamlit pandas
```

Run

```bash
streamlit run app.py
```
